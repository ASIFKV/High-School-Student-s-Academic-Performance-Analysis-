# High-School-Student-s-Academic-Performance-Analysis-
Project overview  A reproducible analysis of the "Students Performance" dataset. The notebook performs exploratory data analysis (EDA), visualizes missing values, inspects distributions and relationships among features, and demonstrates basic modeling and evaluation steps for predicting student performance.

# Dataset

The dataset contains student demographic information and test scores. Typical columns include:

gender

race/ethnicity

parental level of education

lunch

test preparation course

math score

reading score

writing score


# Requirements

Recommended Python environment (tested with Python 3.8+):

pandas

numpy

matplotlib

seaborn

scikit-learn

missingno (optional, for missing-values visualization)
